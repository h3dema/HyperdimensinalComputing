# Hyperdimensional computing (HDC)


HDC or Vector Symbolic Architectures (VSA) is a brain-inspired computing paradigm that represents information using high-dimensional, **random**, and distributed vectors (**hypervectors**, often 10,000+ bits). 
It enables efficient, robust, one-shot learning, and fast parallel processing, making it ideal for edge AI, robotics, and, according to this arXiv paper, low-power cognitive computing. 
Its main components are:
- Hypervectors: Information is encoded into high-dimensional vectors (often 10,000 dimensions or more), which are mostly orthogonal (unrelated) to each other, allowing for robust, holographic representation.
- Operations: HDC uses specific algebraic operations—binding (multiplication), bundling (addition), and permutation—to manipulate these vectors to create complex, compositional data structures.
